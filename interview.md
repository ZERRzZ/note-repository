1.状态码 301 与 302: 301代表永久性转移, 302代表暂时性转移

2.docker命令: 
    docker-compose up -d 启动应用
    docker restart 重启
    docker ps 查看正在运行的容器列表
    docker ps -a 查看正在运行和已停止的
    docker ps | grep 查找
    docker logs -f --tail=要查看末尾多少行
    docker stop 停止容器
    docker rm -f 删除容器
    docker exec 进入容器
